Days between Dates

 

To calculate the number of days between two dates in Excel, subtract the start date from the end date, use DAYS or use DATEDIF.

1. Subtract the start date (A1) from the end date (A2).

2. Use the DAYS function in Excel 2013 or later. The first argument is the end date. The second argument is the start date.

3. Use the DATEDIF function. The first argument is the start date. The second argument is the end date. Fill in "d" for the third argument to get the number of days between two dates.
